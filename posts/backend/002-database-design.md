# 数据库设计基础

数据库是现代应用的核心。良好的数据库设计可以提高应用性能和可维护性。

## 数据库类型

- **关系型**: MySQL、PostgreSQL、SQLServer
- **NoSQL**: MongoDB、Redis、Cassandra
- **时间序列**: InfluxDB、Prometheus

## 关系型数据库设计原则

1. 范式化设计
2. 合理的索引策略
3. 考虑查询性能
4. 备份和恢复计划

## SQL 基础查询

```sql
SELECT * FROM users WHERE age > 18 ORDER BY created_at DESC;
```

掌握数据库基础知识对后端开发至关重要。
