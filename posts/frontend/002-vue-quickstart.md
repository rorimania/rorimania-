# Vue.js 快速上手

Vue 是一个渐进式前端框架，用于构建用户界面。它易学易用，适合各种规模的项目。

## Vue 的优势

- **简单易学**: 温和的学习曲线
- **响应式**: 自动跟踪依赖关系
- **组件化**: 可复用的组件系统
- **灵活**: 既可用于小项目，也可用于大型应用

## Hello World

```vue
<template>
  <div>{{ message }}</div>
</template>

<script>
export default {
  data() {
    return { message: 'Hello Vue!' }
  }
}
</script>
```

开始使用 Vue 构建高效的 Web 应用吧！
