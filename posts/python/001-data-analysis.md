# Python 数据处理与分析

Python 是数据科学和机器学习领域最流行的编程语言。

## 主要库

- **NumPy**: 数值计算
- **Pandas**: 数据处理和分析
- **Matplotlib**: 数据可视化
- **Scikit-learn**: 机器学习

## 数据处理示例

```python
import pandas as pd
import numpy as np

# 创建 DataFrame
df = pd.DataFrame({
    'name': ['Alice', 'Bob', 'Charlie'],
    'age': [25, 30, 35]
})

# 基本统计
print(df.describe())
```
